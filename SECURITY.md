# Security Policy

Security is important to us. FlowScreen is currently in beta; a formal security policy will be published after stable release.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by email:

**hello@hamzauguz.dev**

Do not open public issues for security concerns. Public disclosure before a fix is available can put users at risk.

We will acknowledge receipt and work with you to understand and address the issue. A formal process and response timeline will be defined after stable release.
